##工作性质（work_nature）
1.添加工作性质

url:`work_nature/add`

data: 
```
{
	"name": "他人使用"
}
```
参数说明：

字段名 | 描述 | 参数类型 | 必要 | 备注
:-----: | :-------: | :-------: | :------: | :-------:
name | 工作性质的名字 | str | 1 | 


返回值示例:
```
{
    "code": 0,
    "msg": "成功"
}
```
2.查询所有工作性质的记录

url:`work_nature/listAll`


返回值示例:
```
{
    "code": 0,
    "msg": "成功",
    "data":[
                   {
                       "id": "1",
                       "name": "自用"
                   },
                   {
                       "id": "1540826238787360568",
                       "name": "他人使用"
                   }
               ]
}
```
